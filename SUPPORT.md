# Support

## How To Get Help

Use the channel that best matches your question:

- usage questions: open a GitHub issue with enough context to reproduce your setup and expected behavior
- bug reports: use the bug report issue template
- feature requests: use the feature request template
- security issues: follow [SECURITY.md](SECURITY.md) and avoid public disclosure

## What To Include

Support requests are much easier to answer when they include:

- the package version you are using
- Python version
- whether you are using `cap.core`, `cap.client`, `cap.server`, or a mix
- the relevant request or response payload shape
- a minimal reproducible example
- the full traceback or validation error when applicable
